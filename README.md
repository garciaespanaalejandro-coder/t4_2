
# t4_2

## Descripción:

Proyecto creado para gestionar los pedidos de un bar.

## Tecnologías usadas:
| Tecnología | Descripción |
|---|---|
| Dart | Lenguaje de programación principal |
| Flutter | Framework para el diseño de interfaces |

## Características principales:
- **Separación de responsabilidades:** Arquitectura basada en el patrón **MVVM**.
- **Gestión de pedidos:** Permite asignar pedidos a mesas específicas y gestionar múltiples productos.
- **Cálculo de costos:** Lógica integrada para obtener el total acumulado de forma automática.
- **Interfaz amigable:** Uso de componentes Material Design, Tooltips y validaciones de formularios.
